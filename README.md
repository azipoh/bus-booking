# Moghamo

Moghamo is a modern bus and parcel booking platform for Cameroon. The app lets passengers search trips, book seats, manage reservations, and send or track parcels, while staff and administrators manage branches, buses, schedules, users, and parcel operations.

## Quick start

1. Install dependencies
    - npm install
2. Create environment variables
    - VITE_SUPABASE_URL
    - VITE_SUPABASE_PUBLISHABLE_KEY
    - VITE_GOOGLE_FORM_URL (optional)
3. Start the development server
    - npm run dev

## Useful scripts

- npm run dev — start the Vite development server
- npm run build — create a production build
- npm run test — run the Vitest test suite
- npm run lint — run ESLint
